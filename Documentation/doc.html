Installation on your server via FTP

How To Install MOET Via FTP :
Step 1 - Log into your hosting space via an FTP software (Filezilla for example).
Step 2 - Unzip your Download Pack file and use the extracted MOET theme folder.
Step 3 - Upload the extracted MOET theme folder into one folder or at the root of your server.

Newsletter - Mailchimp / File / Email

1st solution : Mailchimp

Newsletter service

More than 6 million people use MailChimp to create, send, and track email newsletters. Whether you're self-employed, you manage projects for clients, or you work for a Fortune 500 company, MailChimp has features and integrations that will suit your email marketing needs. More informations

For the configuration with Mailchimp, you must just to know your API Key and the ID of your list.

Be sure that the form created in your mailchimp dashboard is asking only the email address and not the name, surname or something else.


  • API Key
  • 1- Click your profile name to open the Account Panel and choose Account.
  • 2- Click the Extras drop-down menu and choose API keys.
  • 3- Copy an existing API key or click the Create A Key button.
  • 4- Label your key to keep your keys organized.
  • 5- Be careful that your list asks just the email address, Signup forms > General forms > Build it. See the screenshot below.


  • List ID
  • 1- Navigate to the Lists page on the top left of your dashboard.
  • 2- Click the list's drop down menu and select Settings.
  • 3- Scroll down to the bottom of the Settings page to find the unique List ID for the list on the bottom left.


Final step

1- Open the file "notify-me.php" findable in the folder php/notify-me.php and fill up the API_KEY and LIST_ID fields.

On line 67 of notify-me.php, you can edit if you want that the subscribers receive a confirmation of their subscription: 

'send_welcome'      => true,

Set true for the confirmation, or false to send nothing
